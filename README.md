# MiQueja
***
## *Plataforma interactiva orientada a la ciudadania usuaria del transporte publico y la recopilacion de sus quejas*
_esta app se enfoca en resolver estos problemas_
---------------------------------
+ Administrar las quejas y comentarios de las personas en el transporte publico
+ generar estadisticas y un sistema de calificaciones del transporte publico
