# MiQueja
***
## *Plataforma interactiva orientada a la ciudadania usuaria del transporte publico y la recopilacion de sus quejas*
<img width="300" height="300" alt="Editing MiQueja_README md at main · CUGDL-UDG_MiQueja - Google Chrome 05_11_2025 11_42_24 a  m  (2)" src="https://github.com/user-attachments/assets/67e945dd-180c-427d-b741-25963ada36b2" />

_esta app se enfoca en resolver estos problemas_
---------------------------------
+ Administrar las quejas y comentarios de las personas en el transporte publico
+ generar estadisticas y un sistema de calificaciones del transporte publico
+ con los datos recopilados tener una linea de comunicacion con el gobierno o responsables del mantenimeinto del transporte publico y generar propuestas a partir
  de las quejas reales de los usuarios.
+ crear una plataforma que sea capaz de brindar a la ciudadania la capacidad de expresar sus quejas y recomendaciones para que se sientan escuchados
